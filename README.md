# SX1255 Dev Hat

A Simple Raspberry Pi Zero shield with the SX1255 RF transceiver chip.  
[`v0.1`](https://github.com/Jankyneering/SX1255_dev_hat/releases/tag/v0.1) based on [M17](https://github.com/M17-Project/SX1255_HAT-hw/blob/main/README.md)'s design, and Semtech's application schematic in the [SX1255 Datasheet](https://www.semtech.com/uploads/documents/sx1255.pdf).

## Production files

Production files for this hat can be found in the [releases](https://github.com/Jankyneering/SX1255_dev_hat/releases). Upload these files to your favorite PCB manufacturer, and make sure to specify the correct board thickness (1.6mm).  
We ordered a batch from JLCPCB and had them assemble the components.

> v0.1 is not intended for production use, but rather as a development platform for testing the SX1255 chip and its software support. It is not optimized for performance, power consumption, RF impedance, and may have some design flaws. Use at your own risk.
